# 🧟 Project Zomboid Server Manager (Windows Edition)

**Project Zomboid Server Manager** — мощное и удобное графическое приложение (GUI) на базе Python и PyQt5 для автоматической установки, настройки, резервного копирования и управления выделенным сервером игры **Project Zomboid** на Windows.

Приложение спроектировано по принципу **«установил и забыл»**: полностью автоматизированы процесс загрузки SteamCMD, установка файлов сервера, проверка порта, отслеживание ошибок, создание бэкапов и работа в системном трее.

## 🌟 Ключевые возможности

### 🛠️ 1. Простота установки и запуска
* **Быстрый старт (`setup.bat`)**: Однокликовый скрипт настройки — проверяет версию Python 3.8+, разворачивает изолированное виртуальное окружение (`venv`), устанавливает все зависимости и автоматически создает ярлык на Рабочем столе.
* **Скрипт запуска (`run.bat`)**: Запускает приложение с автоактивацией виртуального окружения.
* **Автоматическое скачивание и первоначальный запуск**: При первом старте приложения (если сервер не найден) открывается окно автоустановки SteamCMD и самого сервера Project Zomboid (~3 ГБ) в каталог `C:\PZServer`.
* **Автообнаружение имеющихся серверов**: Сканирует популярные пути установки (включая библиотеки Steam на всех дисках) и предлагает использовать уже установленный сервер без повторного скачивания.

### 🎨 2. Современный и удобный GUI
* **Всплывающие подсказки (Tooltips)**: Все кнопки, переключатели, числа и выпадающие списки оснащены русскоязычными подсказками, объясняющими их предназначение.
* **Индикатор прогресса (Progress Bar)**: Отображает точный процент и стадию выполнения при загрузке SteamCMD, распаковке, обновлении сервера и модов.
* **Цветовое выделение логов консоли**:
  * 🟢 **Зелёный**: Успешный запуск сервера ("Server started", "LOGGED IN").
  * 🔴 **Красный**: Ошибки и исключения ("ERROR", "Failed", "Exception").
  * 🔵 **Голубой**: Подключения игроков ("connected", "join").
  * 🟠 **Оранжевый**: Предупреждения ("WARNING").
* **Системный трей (System Tray)**: При закрытии приложение сворачивается возле часов. Быстрый доступ к запуску/остановке сервера через контекстное меню трея и всплывающие уведомления Windows (Toast Notifications).

### ⚙️ 3. "Умная" настройка и пресеты
* **Готовые пресеты сложности Sandbox**: Вкладка Sandbox поддерживает мгновенное переключение пресетов:
  * 🔰 **Начинающий**: Редкие слабые зомби, х3 опыт, много лута.
  * 🏕️ **Выживание**: Стандартный баланс Project Zomboid.
  * 💀 **Хардкор**: Спринтеры, суровая зима, мизерный лут.
  * 🧟 **Орда Zomboid**: Огромное количество зомби, повышенный слух и зрение.
  * 🏗️ **Строитель**: Безопасный мир для строительства и фермерства.
* **Управление бэкапами**: Автоматическое создание бэкапов `servertest.ini` и `SandboxVars.lua` перед изменениями. Удобное диалоговое окно (*Инструменты -> Управление бэкапами*) для восстановления конфигурации в один клик.

### 🛡️ 4. Стабильность и безопасная эксплуатация
* **Глобальный перехват ошибок**: Все неперехваченные сбои записываются в `logs/pz_manager.log` с выводом понятного сообщения пользователю.
* **Проверка свободного места**: Перед обновлением или установкой проверяется наличие минимум 5 ГБ свободного места на целевом диске.
* **Проверка сетевых портов**: Перед запуском проверяются порты `16261`, `8766` и `27015`. Если порты заняты другим приложением, выдается предупреждение.

## 📦 Варианты использования

### Вариант A: Запуск через готовый `.exe` (Без установки Python)
Самый простой вариант для конечного пользователя:
1. Перейдите в папку `dist/PZ Server Manager/`.
2. Запустите файл `PZ Server Manager.exe`.

---

### Вариант B: Быстрая установка из исходников (`setup.bat`)
Если у вас установлен Python 3.8+:
1. Скачайте/клонируйте исходный код приложения.
2. Двойным кликом запустите `setup.bat`.
3. Скрипт создаст `venv`, установит зависимости и создаст ярлык **"PZ Server Manager"** на вашем Рабочем столе.
4. Запускайте приложение через ярлык на Рабочем столе или через `run.bat`.

---

### Вариант C: Сборка `.exe` файла и Windows Инсталлятора

1. **Сборка PyInstaller**:
   ```cmd
   build.bat
   ```
   Результат появится в папке `dist/PZ Server Manager/`.

2. **Сборка Установщика (Inno Setup)**:
   При наличии установленного [Inno Setup 6](https://jrsoftware.org/isinfo.php) откомпилируйте файл `installer.iss`. Вы получите единый файл инсталлятора `installer_output/PZ_Server_Manager_Setup.exe`.

## 📁 Структура проекта

```text
pz-server-manager/
├── main.py                   # Главный файл запуска и глобальный обработчик ошибок
├── main_window.py            # Главное окно GUI, трей, консоль, меню и вкладки
├── config_manager.py         # Менеджер конфигураций (.ini, .lua), бэкапы и пресеты
├── server_installer.py       # Загрузчик SteamCMD, установка сервера, проверка диска
├── server_process.py         # Управление процессом сервера, логирование, проверка портов
├── mod_manager.py            # Модуль работы с модами Steam Workshop
├── translations.py           # Модуль локализации (Русский / English)
├── setup.bat                 # Авто-скрипт создания venv, установки пакетов и ярлыка
├── run.bat                   # Скрипт запуска приложения на Windows
├── build.bat                 # Скрипт сборки PyInstaller и Inno Setup
├── pz_server_manager.spec    # Конфигурационный файл PyInstaller (с фиксом Qt5)
├── installer.iss             # Скрипт Inno Setup для генерации .exe инсталлятора
├── requirements.txt          # Список зависимостей Python (PyQt5, PyInstaller)
├── .gitignore                # Исключения для Git (logs, venv, dist, build)
└── logs/                     # Каталог логов работы приложения (pz_manager.log)
```

## ❓ Часто задаваемые вопросы (FAQ)

#### Вопрос: Нужны ли права Администратора?
*Ответ:* Обычные функции приложения не требуют прав Администратора. Права могут потребоваться только при установке сервера в системные папки (например, `C:\Program Files`).

#### Вопрос: Сервер пишет, что не найдена Java?
*Ответ:* Project Zomboid Dedicated Server требует 64-битную версю Java (JRE/JDK 17+). Если у вас возникла ошибка Java при старте сервера, установите [Adoptium Eclipse Temurin JRE 64-bit](https://adoptium.net/).

#### Вопрос: Как переключить язык интерфейса?
*Ответ:* В меню приложения выберите `Язык (Language) -> 🇷🇺 Русский` или `🇬🇧 English`.
